# Proactive SLA Breach Prediction Using Agentic AI

This project demonstrates a real-time, agentic AI system that predicts and prevents SLA breaches in IT operations.

## Key Features
- Real-time streaming data ingestion using Pathway
- Autonomous AI agents with memory and decision-making
- Event-driven escalation without human prompts
- Production-oriented system architecture

## Tech Stack
- Pathway Engine
- Pathway LLM App
- LangGraph
- Python

## How to Run
1. Start ticket simulator
2. Run Pathway app
3. Observe agent-driven escalation in real time
